xyCppUtils
==========

Cpp utilities by Ying Xiong.

Author: Ying Xiong.  
Created: Aug 09, 2012.


Compile and install
-------------------

    cd /path/to/xyCppUtils
    mkdir cmake-build
    cd cmake-build
    cmake [-DCMAKE_BUILD_TYPE=RELEASE] ..
    make
    make test  # optional
    [sudo] make install
